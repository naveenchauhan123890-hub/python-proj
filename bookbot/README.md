# bookbot

BookBot is my first [Boot.dev](https://www.boot.dev) project!

It analyzes a book and prints its word count and a sorted character count.

## Run

```bash
python3 main.py books/frankenstein.txt
```

## Submit / run the Boot.dev lessons

Run these from **inside this `bookbot/` folder**. Add `-s` to submit instead of
just running:

```bash
bootdev run 4695fb05-cb7c-44e4-b206-0ed8eac44588 -s   # lesson 1
bootdev run 5be3e3bd-efb5-4664-a9e9-7111be783271 -s   # lesson 2
```
